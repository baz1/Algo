std::vector<std::string> split(const std::string & haystack, const char needle, int min_capacity = 0)

Splits a string.

Splits the string haystack with the separator needle, and returns the result as a string vector which members may be empty. If min_capacity is non-zero, it will be used as the initial capacity for the resulting vector.
