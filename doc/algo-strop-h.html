std::vector<std::string> split(const std::string haystack, const char needle, int min_capacity = 0)

Splits a string.

Splits the string haystack with the separator needle, and returns the result as a string vector which members may be empty. If min_capacity is non-zero, it will be used as the initial capacity for the resulting vector.

Note: This function is linear in the length of haystack.

std::vector<std::string> split(const std::string haystack, const std::string needle, int min_capacity = 0)

Splits a string.

Splits the string haystack with the separator needle, and returns the result as a string vector which members may be empty. If min_capacity is non-zero, it will be used as the initial capacity for the resulting vector.

Note: This function operates at most in O(nm) where m and n are the length of haystack and needle.

bool starts_with(const std::string main_str, const std::string beginning)

Returns true if main_str starts with beginning, false otherwise.

Note: This function is linear in the length of beginning.

Note: This function is only a specialized version of the starts_with on iterable classes.
