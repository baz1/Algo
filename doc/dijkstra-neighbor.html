Detailed Description

Neighbor of a vertex.

This class is to be used alongside the dijkstra procedure, and defines the neighbor of a vertex.

See also <algo/dijkstra.h>.

Member Function Documentation

dijkstra_neighbor::dijkstra_neighbor(int target, WeightType weight)

Constructs a neighbor element.

Its neighbor identifier shall be target whereas the cost of the connection is weight.

Member Variable Documentation

int dijkstra_neighbor::target

The target vertex.

This integer defines the identity of the neighbor at the end of the connection.

WeightType dijkstra_neighbor::weight

The weight of the connection.

This weight measures the cost of the connection used when searching the shortest path in Dijkstra's algorithm.
